Criminalidad en España
PowerBi Dashboard

PowerBi dashboard where you could find data about crimes in Spain from 2013 to 2021

This project has been made using python for the data ingestion and data cleaning. Once the data has been processed the dataframe has been taken into the power bi app using the power bi python script functionality and then we have made the visualizations. After that we have created a panel for sharing with stakeholders if needed, and lastly we have embedded the project here. The data has been collected from www.ine.es source.
