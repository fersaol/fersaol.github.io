AWS React-Redux Coffee Shop Web

This is a dynamic website made using react and redux for handling an online coffee shop. The architecture is a 3 tier web in a cloud infrastructue as follows:

  • Amplify: delivers the web page to the client
  • Api Gateway: routes the client requests to the appropriate lambda function
  • Lambda Functions: perform the computational work and the business logic updating/getting the data from the dynamodb
  • dynamodb: stores the store products
Github allows us for CI/CD and redux along with redux/toolkit allows us to manage the cart, storing the products temporarily, until the purchase is confirmed and then it clears the store and make the changes needed onto the dynamodb, permanently.

Note: In progress

  • Main Tools used: Amplify, Dynamodb, Lambda, Api Gateway Github, IAM, Redux/toolkit and Redux, React, JavaScript
  • Category: Cloud, Web Development
  • Year: April 2024
