Inflaction Dashboard

The dashboard has been created to acknowledge in a visual and easy way the inflaction effect and why it is so damaging for the personal economy.

The project has been developed using streamlit and python and made available for the public using the AWS ifrastructure as follows:

  • Route 53: makes the access easier to the client by providing the domain name
  • ECS (Elastic Conatiners Service): service that allows the deployment of containers
  • Docker Hub: service that contains images for deploying containers
  • Fargate: serverless compute service for ECS that makes the web app scalable and cost effective
  • S3: Object storage service that holds the dataset for the Spain's public debt
  • Inflaction dataset: retrieved through a GET request to the INE (Instituto Nacional de Estadística) api
