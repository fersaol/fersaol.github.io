Customer Churn ML

Exploratory Data Analysis and hypothesis testing to see if some of the variables would be worthy to mantain. Once this is done, we use some supervised Machine learning algorithms to classify the customers in churn and not churn. Then we show the data using a Dash dashboard where you can find your own insights and take a look at the data and a Streamlit app where you can make your own predictions and get insights with a fully functional machine learning algorithm.

  • Main Tools used: Python, Scikit-learn, Plotly, Pandas, Numpy, Dash, Streamlit
  • Category: Classification, Prescriptive analysis, Predictive analysis
  • Year: March 2024
