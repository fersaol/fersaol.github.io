World Co2 Emissions ML

Machine Learning project that ensambles three models in order to obtain a web app that allows countries to predict its efficiency producing energy so they can be classified in a cluster and get recommendations with the aim of checking their environmental policies.

  • Main Tools used: Python, Scikit-learn, Plotly, Pandas, Numpy, Flask, Streamlit
  • Category: Clustering, Regression, Classification
  • Year: October 2022
